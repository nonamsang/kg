### 🌱 개요
- 사용자가 환경인증 기업에 등록된 상품을 결제하면 해당 상품의 **탄소 배출량**을 계산하여 **에코 포인트** 적립  
- 적립된 에코 포인트를 통해 **개인의 친환경 소비 활동** 시각화 및 관리  
- 환경인증 기업의 판매 데이터와 탄소 절감 효과를 분석하여 **기업 인센티브** 제공

---

### 💡 기획 의도

| 목적 | 설명 |
|------|------|
| **편의성 향상** | 현대적인 웹 기반 플랫폼 기능 구현으로 쉽고 빠른 친환경 소비 환경 제공 |
| **탄소 배출 인식** | 2026년부터 금융회사까지 온실가스 배출량("금융배출량") 공시 시대 도래, 기업과 개인이 탄소 배출을 직관적으로 인지 |
| **개인 맞춤 탄소 관리** | 카드 사용 내역 기반 **탄소 가계부**와 **그린 포인트 보상 시스템** 제공으로 친환경 소비와 저탄소 생활 습관 유도 |
| **기업 ESG 활동 지원** | 기업의 친환경 활동과 ESG 기반 투자 공시 및 홍보 채널 제공 |
| **환경적 지속가능성 강화** | 개인과 기업 모두가 탄소중립 달성에 기여할 수 있는 디지털 기반 그린핀테크 플랫폼 구현 |

- 디지털 기술 발전을 활용하여 개인과 기업 모두 **탄소 감축 효과**를 실생활에서 직관적으로 체감 가능  
- 금융 생태계와 대중 간의 **투명한 연결**을 통해 친환경 소비와 저탄소 전략 실행을 동시에 지원
